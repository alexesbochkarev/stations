
В терминале из корневой папки, где находится файл requirements.txt выполнить команду: "pip install -r requirements.txt"

1. В папке /stations в файле settings.py отредактировать настройки подключения к базе данных (87 строка)
2. В корневой директории проекта (там где файл manage.py) в терминале выполнить команду: "python manage.py migrate"
3. В корневой директории проекта (там где файл manage.py) в терминале выполнить команду: "python manage.py collectstatic"
4. В корневой директории проекта (там где файл manage.py) в терминале выполнить команду: "python manage.py createsuperuser" далее указать логин и пароль для суперадмина
5. В корневой директории проекта (там где файл manage.py) в терминале выполнить команду: "python manage.py runserver 127.0.0.1:8000" можно запустить на любом другом порту
Рекомендую запускать приложение с помощью pm2 https://pm2.keymetrics.io
6. В файле mediafiles.txt пример конфига nginx для корректной работы медиафайлов после настройки в файле settings.py на строке 26 установить debug в значение False



http://ip адрес:порт/admin/  Админка (логин и пароль суперпользователя)
http://ip адрес:порт/api/stations/ Список станций
http://ip адрес:порт/api/stations/<id>/ Детализация станции по id
